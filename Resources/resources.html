Array

1. What is Array ?

An array is a collection of similar data elements stored at contiguous memory locations. It is the simplest data structure where each data element can be accessed directly by only using its index number.

2. Need of using Array ?

In programming, most of the cases need to store a large amount of data of a similar type. We need to define numerous variables to store such a huge amount of data. Linked List

1. What is Linked List ?

A linked list is a linear data structure that includes a series of connected nodes. Here, each node stores the data and the address of the next node. You have to start somewhere, so we give the address of the first node a special name called HEAD. Also, the last node in the linked list can be identified because its next portion points to NULL.

Read the full article

2. Need of using Linked List ?

Linked lists can be used to represent polynomials. Using a linked list, we can perform the polynomial manipulation. Arithmetic operations like addition or subtraction of long integers can also be performed using a linked list. The linked list can be used to implement stacks and queues.
